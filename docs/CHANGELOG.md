# BearCore-V 更新日誌

所有重要的變更都會記錄在這個文件中。

## [1.0.0] - 2024-01-XX

### 新增
- 完整的 5級流水線 RISC-V RV32IM 處理器
- 硬體乘法器和除法器
- UART 串口輸出 (115200 波特率)
- 性能計數器 (週期數、指令數、CPI)
- Python 指令級模擬器 (Golden Model)
- 完整的測試套件

### 改進
- 資料前推 (Forwarding) 機制優化
- Load-Use 衝突檢測
- 記憶體對齊存取處理
- 專案結構重構和清理

### 修復
- 除法除零處理
- 分支目標計算
- UART 狀態機穩定性

### 技術規格
- 指令集: RV32IM
- 流水線: 5級 (IF/ID/EX/MEM/WB)
- 記憶體: 16KB ROM + 64KB RAM
- 性能: CPI = 1.0 (理想流水線)
